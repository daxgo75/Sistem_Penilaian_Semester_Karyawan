# Sistem Penilaian Semester Karyawan

## Deskripsi

Sistem Penilaian Semester Karyawan adalah aplikasi berbasis web yang dikembangkan menggunakan Laravel 10. Aplikasi ini dirancang untuk membantu perusahaan dalam melakukan evaluasi kinerja karyawan setiap semester secara efisien, objektif, dan terstruktur.  

Sistem ini menyediakan fitur pengelolaan data karyawan, pemberian nilai berdasarkan kriteria tertentu, serta pembuatan laporan hasil penilaian dalam format PDF dan Excel yang dapat difilter berdasarkan divisi dan semester.  

---

## Fitur Utama

- Manajemen data karyawan dan penilaian per semester  
- Filter data berdasarkan divisi dan semester  
- Export data ke format PDF dan Excel  
- Menampilkan tanda tangan digital General Manager pada laporan  
- Penyimpanan dan pencetakan laporan per divisi  
- Pencarian dan filter data dinamis  

---

## Tujuan Pengembangan

Proyek ini dibuat untuk mendukung proses evaluasi kinerja karyawan agar lebih:
- Terukur dan objektif  
- Cepat dan akurat  
- Mudah diakses oleh HRD dan manajemen  
- Mengurangi risiko kesalahan dalam proses penilaian manual  

---

## Teknologi yang Digunakan

| Komponen | Teknologi |
|-----------|------------|
| Framework | Laravel 10 |
| Frontend | Tailwind CSS, Bootstrap 4 |
| Database | MySQL |
| Export PDF | DomPDF |
| Export Excel | Maatwebsite Excel |
| Authentication | Laravel Breeze |
| Server Lokal | Laragon |
